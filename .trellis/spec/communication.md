# 沟通规范

> 所有沟通使用中文。

---

## 语言要求

所有文档、代码注释、Git 提交信息、PR 描述、Issue 讨论必须使用中文。AI 助手回复也必须使用中文。

---

## Git 提交格式

```
type(scope): 可保留英文专业用词的中文提交信息
```

- `type`：英文小写（`feat` / `fix` / `refactor` / `docs` / `test` / `chore`）
- `scope`：英文小写，写领域或模块（如 `hook` / `workflow` / `frontend` / `executor`）
- 冒号后中文描述，聚焦"做成了什么"
- 一个提交只表达一个主要意图
