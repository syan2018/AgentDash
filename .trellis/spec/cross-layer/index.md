## 归属原则

以下类型的文档应放在此目录：

- 前后端共享的 JSON 序列化契约（字段命名、结构定义）
- 跨 binary（云端/本机）的通信协议
- 前端消费后端事件流的格式约定

如果一个文档**主要**服务于某一端的开发者（即使涉及另一端），应放在对应 layer 目录下。
